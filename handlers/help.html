Basic Arithmetic Operations

The calculator supports standard arithmetic operations:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Modulo: %
  • Exponentiation: ^

Example: 2 + 3 * 4 - 5 / 2

Numbers

The calculator supports various number formats:

  • Decimal: 3.14
  • Integer: 42
  • Scientific notation: 6.022e23
  • Binary (base 2): Prefix with 0b
  • Octal (base 8): Prefix with 0
  • Hexadecimal (base 16): Prefix with 0x or 0X

Example: 0b1010 (equals 10 in decimal), 012 (equals 10 in decimal), 0xA or 0XA (equals 10 in decimal)

Mathematical Constants

Built-in mathematical constants:

  • Pi: pi
  • Euler's number: e

Example: pi * r^2

Functions

The calculator includes many built-in mathematical functions:

  • Trigonometric: sin(), cos(), tan(), asin(), acos(), atan()
  • Logarithmic: log() (base 10), ln() (natural log)
  • Exponential: exp()
  • Square root: sqrt()
  • Absolute value: abs()
  • Rounding: floor(), ceil(), round()

Example: sqrt(16) + log(10), sin(pi/2)

Variables and Assignments

You can define and use variables in expressions:

  • Assignment: Use =
  • Variable names can contain letters, numbers, and underscores, but must start with a letter or underscore

Example: x = 5; y = x + 3; y * 2

Complex Expressions

You can combine various operations and functions to create complex expressions:

Example: 2 * sin(x) + sqrt(y^2 - 4*z)

Parentheses

Use parentheses to group parts of expressions and control the order of operations:

Example: (2 + 3) * 4

Comparisons and Logic

The calculator supports comparison and logical operations:

  • Equality: ==
  • Inequality: !=
  • Greater than: >
  • Less than: <
  • Greater than or equal to: >=
  • Less than or equal to: <=
  • AND: &&
  • OR: ||
  • NOT: !

Example: x > 0 && y < 10

Function Definitions

You can define custom functions:

  // function body
  return expression
}

Example:

  return x * x
}

Example of a recursive function:

def factorial(n) {
  if (n <= 1) {
    return 1
  } else {

Precision

The calculator supports high precision calculations up to 100,000 digits.

Order of Operations

The calculator follows standard mathematical order of operations (PEMDAS):

  1. Parentheses
  2. Exponents
  3. Multiplication and Division (left-to-right)
  4. Addition and Subtraction (left-to-right)

Error Handling

The calculator will return an error message for invalid expressions, such as:

  • Division by zero
  • Invalid function calls
  • Syntax errors
  • Undefined variables or functions
